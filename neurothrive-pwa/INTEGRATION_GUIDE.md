# Integration Guide: Job Search Skills

## How the Two Skills Work Together

This guide explains how to use the **Job Search Skill** and **Neurodivergent Job Search Skill** together for maximum effectiveness.

---

## Quick Reference

**Job Search Skill** = Foundation (core job search best practices)  
**Neurodivergent Job Search Skill** = Enhancement Layer (executive function + mental health support)

**Together** = Comprehensive neurodivergent-friendly job search system

---

## When to Use Which Skill

### Use Job Search Skill Alone When:
- You want standard resume review
- You need interview preparation
- You're researching companies
- You want salary negotiation help
- You need cover letter guidance
- Executive function is not a barrier for you today

### Use Both Skills Together When:
- You're neurodivergent and struggle with executive function
- You need mental health support during job search
- You want automated job aggregation and filtering
- Overwhelm is preventing you from starting
- You need energy and mood tracking
- You want ND-specific company research

### The Power of Integration

**Example: Resume Review**

**Job Search Skill Only:**
- Reviews resume as senior recruiter
- Identifies strengths and weaknesses
- Provides specific improvements
- Company research (if requested)
- Tailored resume creation

**+ Neurodivergent Skill:**
- ALL of the above, PLUS:
- Checks for ND-friendly indicators at company
- Flags sensory/environment concerns
- Suggests disclosure strategy
- Provides emotional support about imposter syndrome
- Tracks your energy when doing this task
- Breaks resume work into manageable chunks

### Interview Preparation

**Job Search Skill provides:**
- STAR method framework
- Question anticipation
- Company-specific preparation
- Thank-you email templates

**Neurodivergent Skill adds:**
- Sensory planning (what to wear, bring)
- Accommodation request guidance
- Energy management for interview day
- Post-interview emotional support
- Disclosure timing strategies

**Integrated Workflow:**
1. User has interview scheduled (tracked by ND Skill)
2. 3 days before: Trigger prep reminder (ND Skill)
3. Generate STAR stories from resume (Job Search Skill)
4. Anticipate questions based on company research (Job Search Skill)
5. Plan sensory accommodations (ND Skill: headphones, quiet room request)
6. Prepare disclosure talking points if applicable (ND Skill)
7. Day-of energy check-in and pacing advice (ND Skill)
8. Post-interview debrief and emotional support (ND Skill)

### Application Tracking

**Job Search Skill provides:**
- Status tracking framework
- Follow-up timing guidance
- Metrics to monitor (response rate, interview rate)

**Neurodivergent Skill adds:**
- Automatic tracking (zero user effort)
- Visual status indicators
- Working memory support (context restored)
- Emotional pattern tracking (mood after rejection)
- Energy-aware pacing recommendations

**Integrated System:**
- Applications automatically logged with ALL context
- Follow-ups suggested but not pressured
- Rejection emotions validated and reframed
- Success patterns identified ("You get interviews when you apply to remote roles")
- Pacing adjusted based on energy and mood

## Prompt Templates Combining Both Skills

### Setup Prompt (First Time)

```
I'm a neurodivergent job seeker with [ADHD/autism/bipolar/etc.] looking for [JOB TITLE] roles in [LOCATION/REMOTE].

Please use BOTH the Job Search Skill and Neurodivergent Job Search Skill.

My executive function challenges:
[X] Task initiation - hard to start
[X] Decision-making - paralysis by analysis
[X] Working memory - forget what I've done
[X] Time management - hours disappear
[X] Emotional regulation - rejection hits hard
[X] Energy management - fluctuates daily

My current state:
- Mood: [1-5 or description]
- Energy: [low/moderate/high]
- Job search status: [just starting/active/burned out/etc.]

Help me:
1. Set up automated job search with ND-friendly filtering
2. Configure morning reports that won't overwhelm me
3. Understand my resume review prompt system
4. Start mood tracking to learn my patterns

[Attach resume]
```

### Resume Review Prompt (Combining Skills)

```
I want to apply to [JOB TITLE] at [COMPANY].

Using BOTH Job Search Skill and Neurodivergent Job Search Skill:

1. Research this company:
   - Standard info (size, stage, tech stack, culture)
   - ND-specific indicators (flexibility, accommodations, sensory environment)

2. Build recruiter persona for THIS company

3. Analyze my fit:
   - Do I meet requirements?
   - What excites them about me?
   - What concerns might they have?
   - ND-specific: Is this a good environment for my needs?

4. Disclosure strategy:
   - Should I mention my [ADHD/autism/etc.]?
   - When? (Cover letter, interview, post-offer?)
   - How to frame it as strength?

5. Tailor my resume for THIS role at THIS company

6. Emotional support:
   - Reality-check my imposter syndrome
   - Remind me of my qualifications
   - Encourage me to apply

[Paste job posting or URL]
[Attach resume]
```

### Pattern Analysis Prompt (ND Skill Focus)

```
I've been job searching for [TIME PERIOD] using your system.

Based on my mood tracking and application data:

1. What patterns do you see?
   - When am I most/least productive?
   - How does mood correlate with applications?
   - What triggers anxiety or low energy?

2. What's working well in my approach?

3. What should I adjust?
   - Pacing (too fast/slow?)

   - How many applications sent?
   - Interview rate?
   - Am I on track or being too hard on myself?

```
